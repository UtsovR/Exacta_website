Exacta Web Solution

Exacta Web Solution is a modern, conversion-focused digital marketing website built using React, TypeScript, Tailwind CSS, and Framer Motion. The project is designed with a premium dark aesthetic, cinematic hero video, floating dynamic navigation bar, and WhatsApp-integrated call-to-action system.

This repository contains the full source code of the official website.

Project Overview

The website is designed to present Exacta Web Solution as a professional digital marketing agency offering:

Social Media Marketing

Custom Website Development

SEO and Performance Marketing

The focus of the project is performance, modern UI/UX, smooth animations, responsiveness, and strong conversion-oriented design.

Key Features

Cinematic looping hero background video

Floating, dynamic island-style navigation bar

Glassmorphism UI elements

Fully responsive layout across devices

WhatsApp “Chat With Us” integration

Installation and Setup
Clone the repository
git clone https://github.com/your-username/exacta-web-solution.git

Navigate to the project directory
cd exacta-web-solution

Install dependencies
npm install

Run development server
npm run dev


The project will run locally at:

http://localhost:5173

Production Build

To create a production build:

npm run build


To preview the production build:

npm run preview
